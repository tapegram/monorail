# Web Stack: PicoCSS (Classless) + htmx

Claude should use:

- PicoCSS docs: https://picocss.com/docs/
- htmx docs: https://htmx.org/docs/

Rules:

- Pages must use semantic HTML.
- `full` page includes PicoCSS classless version + htmx headers.
- `partial` page returns only the inner HTML.
- Use htmx attributes (hx-get, hx-post, hx-trigger, hx-swap, hx-target).
- **Use the classless version of Pico CSS** which automatically styles semantic HTML without requiring CSS classes.
- Avoid adding CSS classes unless absolutely necessary for custom styling.
- Rely on semantic HTML elements (`<main>`, `<article>`, `<section>`, `<nav>`, etc.) for proper styling.
