# generate-page-and-route

## Purpose

Add a new page + controller + route.

## Behavior

1. Add controller in app.controllers.
2. Add page in app.pages.
3. Add route in app.routes using Route ability.
4. Use htmx attributes per https://htmx.org/docs/.
5. Use semantic HTML that works with PicoCSS classless version (no CSS classes needed).
6. Typecheck.
